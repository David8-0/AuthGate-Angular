Developers

1. Sign In with Authgate Button

Create a "Sign In with Authgate" button in your Angular frontend application. Clicking it should send an HTTP request to 'Authgate_Backend_Url/tenants/authorize-client' with the Client_ID and Client_Secret to verify the tenant.

2. Redirect to Authgate Login Page

You will receive the Authgate login page URL with the projectID appended to it. Then, you need to append a Code Challenge to implement the PKCE flow and redirect to it.

3. Adjust app.routes.ts File

Modify your app.routes.ts file by adding an extra route that matches the callback URL with '/:authCode' appended to it.

4. Adjust Component Used in Callback URL

Modify the component that you are redirecting to in the app.routes.ts file to extract the authentication code and send it to your backend server along with the code verifier.

Response

5. Exchange Authorization Code and Code Verifier for Authentication Token

Your backend server should send a request to Authgate and receive an authorization token in return. Extract the user information from the token, save it in your database, and issue a JWT to be sent to your Angular frontend application.

6. Maintain User State

Save the token in your Angular frontend application and maintain user state.
